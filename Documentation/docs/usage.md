# Usage

# Mac OS X

You'll need the following libraries to build in OS X (all of them can be install with `brew install {library}`)

1. premake
2. assimp
3. jsoncpp

# Linux

You'll need the following libraries to build in Linux

0. premake
1. jsoncpp
2. yaml-cpp
3. assimp
4. vulkan-extra-layers
5. vulkan-extra-tools
